# Интеграция с Raydium API для Devnet

## Описание

Компонент `SwapPage.vue` был обновлен для интеграции с реальным API Raydium в сети devnet. Теперь свапы происходят через официальный API Raydium с реальными котировками и транзакциями.

## Основные возможности

### 🔄 Реальные свапы через Raydium
- Интеграция с официальным API Raydium devnet
- Реальные котировки в реальном времени
- Автоматическое создание транзакций
- Поддержка всех токенов доступных в Raydium devnet

### 💰 Котировки в реальном времени
- Автоматическое получение котировок при изменении суммы или токенов
- Debounce для оптимизации API запросов (500ms)
- Отображение price impact, комиссий и минимальной суммы получения
- Цветовая индикация price impact (зеленый/желтый/красный)

### 🎯 Улучшенный UX
- Индикация загрузки котировок
- Отображение провайдера (Raydium)
- Реальные комиссии сети
- Автоматическое обновление суммы получения

## API Endpoints

### Raydium Trade API
- **Base URL**: `https://transaction-v1.raydium.io`
- **Quote**: `GET /compute/swap-base-in` - получение котировки для свапа
- **Swap**: `POST /transaction/swap-base-in` - создание транзакции свапа

### Параметры Quote API
```javascript
{
  inputMint: string,      // Адрес токена для продажи
  outputMint: string,     // Адрес токена для покупки
  amount: string,         // Количество в минимальных единицах (с учетом decimals)
  slippageBps: string,    // Проскальзывание в базисных пунктах (100 = 1%)
  txVersion: string       // Версия транзакции ('V0' для versioned)
}
```

### Параметры Swap API
```javascript
{
  computeUnitPriceMicroLamports: string, // Приоритетная комиссия
  swapResponse: object,   // Ответ от quote API
  txVersion: string,      // Версия транзакции
  wallet: string,         // Публичный ключ пользователя
  wrapSol: boolean,       // Обернуть SOL в wSOL
  unwrapSol: boolean,     // Развернуть wSOL в SOL
  inputAccount: string,   // Адрес входного аккаунта (опционально)
  outputAccount: string   // Адрес выходного аккаунта (опционально)
}
```

## Основные функции

### `getRaydiumQuote(fromToken, toToken, amount)`
Получает котировку от Raydium API для указанных токенов и суммы.

### `createRaydiumSwapTransaction(quote, userPublicKey)`
Создает транзакцию свапа на основе полученной котировки.

### `loadRaydiumTokens()`
Загружает список доступных токенов из Raydium API.

### `getQuote()`
Основная функция для получения котировки с debounce и обработкой ошибок.

## Состояния компонента

### Reactive состояния
- `currentQuote` - текущая котировка от Raydium
- `isGettingQuote` - индикатор загрузки котировки
- `quoteTimeout` - timeout для debounce

### Computed свойства
- `exchangeRate` - курс обмена на основе котировки
- `minimumReceived` - минимальная сумма получения с учетом slippage
- `showSwapDetails` - показывать ли детали свапа

## Обработка ошибок

### Типы ошибок
1. **API недоступен** - fallback на базовые токены
2. **Ошибка котировки** - очистка суммы получения
3. **Ошибка создания транзакции** - уведомление пользователя
4. **Ошибка подписи** - детальная обработка ошибок Phantom

### Fallback механизмы
- При недоступности Raydium API используются базовые токены
- При ошибке котировки показывается пустая форма
- При ошибке транзакции предлагается повторить

## Интеграция с Phantom Wallet

### Проверка сети
- Автоматическая проверка подключения к devnet
- Предупреждение при несоответствии сети
- Возможность продолжить с предупреждением

### Подпись транзакций
- Использование `signAndSendTransaction` (основной метод)
- Fallback на `signTransaction` + `sendRawTransaction`
- Обработка всех типов ошибок Phantom
