# Интеграция Phantom Wallet с базой данных пользователей

## Описание

Компонент `PhantomWalletButton.vue` был обновлен для автоматического создания пользователей в базе данных при подключении Phantom Wallet.

## Функциональность

### Автоматическое создание пользователей
- При подключении Phantom Wallet автоматически создается запись пользователя в базе данных
- Генерируется случайный username на основе адреса кошелька
- Обрабатываются случаи когда пользователь уже существует

### Генерация username
- Формат: `user_{shortAddress}_{randomSuffix}`
- Пример: `user_1234567890_1234`

### API Endpoint
- **URL**: `https://launchpad-wl8n.onrender.com/api/users`
- **Method**: `POST`
- **Body**:
```json
{
  "wallet_address": "string",
  "username": "string", 
  "avatar_url": "string|null"
}
```

### Обработка ответов
- **Успех**: Пользователь создан или найден
- **Конфликт**: Пользователь уже существует (код 409)
- **Ошибка**: Логируется в консоль, но не прерывает подключение кошелька

## Состояния компонента

### Новые переменные
- `userInfo` - информация о пользователе из БД
- `isCreatingUser` - статус создания пользователя

### UI обновления
- Кнопка подключения показывает "Creating user..." во время создания
- Отображается username пользователя в подключенном состоянии
- Кнопка отключается во время создания пользователя

## Логирование

Все операции логируются в консоль:
- `👤 Создание пользователя в базе данных`
- `✅ Пользователь создан/найден в базе данных`
- `ℹ️ Пользователь уже существует в базе данных`
- `❌ Ошибка создания пользователя`

## Безопасность

- Ошибки БД не прерывают подключение кошелька
- Валидация данных на сервере
- Обработка сетевых ошибок

## Тестирование

1. Подключите Phantom Wallet
2. Проверьте консоль браузера на наличие логов
3. Убедитесь что пользователь создан в базе данных
4. Проверьте отображение username в интерфейсе
