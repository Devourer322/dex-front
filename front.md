# Документация фронтенда DEX приложения

## Обзор архитектуры

DEX приложение построено на Vue.js 3 с использованием Composition API и Vue Router. Приложение представляет собой децентрализованную биржу с функционалом создания токенов, торговли и управления профилями пользователей.

## Структура проекта

```
dexfront/
├── src/
│   ├── components/          # Vue компоненты
│   ├── router/             # Маршрутизация
│   ├── assets/             # Статические ресурсы
│   ├── App.vue             # Главный компонент
│   ├── main.js             # Точка входа
│   └── style.css           # Глобальные стили
├── public/                 # Публичные файлы
├── package.json            # Зависимости
└── vite.config.js          # Конфигурация Vite
```

## Маршрутизация

### Основные маршруты

| Маршрут | Компонент | Описание |
|---------|-----------|----------|
| `/` | Dashboard | Главная страница (редирект) |
| `/dashboard` | Dashboard | Дашборд с токенами |
| `/profile/:address` | UserProfile | Профиль пользователя |
| `/profile` | UserProfile | Профиль по умолчанию |
| `/trading` | SwapPage | Страница обмена токенов |
| `/staking` | TrademarkGuidelines | Руководство по торговым маркам |
| `/create-token` | CreateTokenPage | Создание нового токена |
| `/coin/:mint_address` | TokenChart | Страница токена с графиком |
| `/settings` | SettingsPage | Настройки приложения |

## Компоненты

### 1. App.vue - Главный компонент

**Расположение элементов:**
- **Мобильное меню** (слева вверху, только на мобильных)
- **NavigationPanel** (слева, фиксированная панель)
- **PhantomWalletButton** (справа вверху, фиксированная)
- **Основной контент** (центр, с отступом слева)

**Адаптивность:**
- Desktop: боковая панель 280px
- Tablet: боковая панель 260px
- Mobile: скрытая панель с оверлеем

### 2. NavigationPanel.vue - Панель навигации

**Расположение элементов:**
- **Логотип** (вверху)
- **Навигационные ссылки** (центр):
  - Dashboard
  - Profile (динамический путь)
  - Swap
  - Trademark Guidelines
  - Settings
- **Кнопка создания токена** (внизу)

**Особенности:**
- Динамический путь профиля на основе подключенного кошелька
- Активное состояние для текущего маршрута
- Адаптивная ширина

### 3. PhantomWalletButton.vue - Кнопка кошелька

**Расположение элементов:**
- **Кнопка подключения** (когда не подключен)
- **Информация о кошельке** (когда подключен):
  - Аватар Phantom
  - Сокращенный адрес
  - Username (если есть)
  - Баланс SOL
  - Кнопка отключения

**Функциональность:**
- Автоматическое подключение при загрузке
- Создание пользователя в базе данных
- Получение баланса SOL

### 4. Dashboard.vue - Главная страница

**Расположение элементов:**
- **TokenSearch** (поиск токенов)
- **TrendingTokens** (популярные токены)
- **AllTokens** (все токены)

**Состояния:**
- Загрузка (спиннер)
- Ошибка (с кнопкой повтора)
- Контент (список токенов)

### 5. TokenSearch.vue - Поиск токенов

**Расположение элементов:**
- **Поле поиска** (с иконкой)
- **Кнопка поиска**
- **Результаты поиска** (выпадающий список)

**Функциональность:**
- Поиск по названию и тикеру
- Переход к странице токена при выборе

### 6. TrendingTokens.vue - Популярные токены

**Расположение элементов:**
- **Заголовок** с кнопками навигации
- **Горизонтальная прокрутка** карточек токенов

**Карточка токена:**
- Аватар токена
- Название и тикер
- Market cap и replies
- Описание

### 7. AllTokens.vue - Все токены

**Расположение элементов:**
- **Вкладки**: explore/watchlist
- **Фильтры**: сортировка и статус
- **Сетка токенов** (адаптивная)

**Карточка токена:**
- Изображение токена
- Метаданные (создатель, время)
- Статистика (market cap, replies)
- Название и описание

### 8. SwapPage.vue - Страница обмена

**Расположение элементов:**
- **Кнопка настроек** (справа вверху)
- **Форма обмена**:
  - Секция продажи (токен + сумма)
  - Кнопка переключения токенов
  - Секция покупки (токен + сумма)
  - Кнопка обмена/подключения кошелька
  - Детали обмена (курс, комиссии)

**Модальные окна:**
- **Настройки**: slippage, скорость, защита
- **Выбор токена**: поиск и список токенов

**Интеграция:**
- Raydium API для получения котировок
- Phantom Wallet для подписи транзакций

### 9. CreateTokenPage.vue - Создание токена

**Расположение элементов:**
- **Заголовок страницы**
- **Форма создания**:
  - Название и тикер токена
  - Описание
  - Социальные ссылки (раскрывающиеся)
  - Загрузка изображения
  - Загрузка баннера (опционально)
  - Предупреждение о неизменяемости
  - Кнопка создания

**Функциональность:**
- Drag & drop для изображений
- Валидация формы
- Интеграция с Phantom Wallet
- Создание токена через API

### 10. TokenChart.vue - Страница токена

**Расположение элементов:**
- **Заголовок** с кнопкой "назад"
- **Информация о токене**:
  - Аватар и название
  - Метаданные (статус, market cap, replies)
  - Статус WebSocket соединения
- **Основной контент**:
  - **Левая панель**: график с контролами
  - **Правая панель**: информация о токене

**Контролы графика:**
- Временные интервалы (1m, 5m, 15m, 1h, 4h, 1d)
- Настройки отображения
- Кнопки управления

**Информационная панель:**
- Цена и изменения
- Market cap и объем
- Социальные ссылки
- Статистика

### 11. UserProfile.vue - Профиль пользователя

**Расположение элементов:**
- **Основная панель** (слева):
  - Аватар и информация
  - Статистика (followers, following, created coins)
  - Вкладки (balances, replies, notifications)
  - Контент вкладок
- **Боковая панель** (справа):
  - "Кого читать" с предложениями

**Функциональность:**
- Загрузка профиля по адресу кошелька
- Копирование адреса
- Ссылка на Solscan
- Отображение балансов

### 12. SettingsPage.vue - Настройки

**Расположение элементов:**
- **Заголовок страницы**
- **Секции настроек**:
  - Тема (темный режим)
  - Уведомления
  - Сеть (devnet/mainnet)

### 13. TrademarkGuidelines.vue - Руководство по торговым маркам

**Расположение элементов:**
- **Заголовок** с датой утверждения
- **Содержание**:
  - Введение
  - Руководящие принципы
  - Сообщение о нарушении
  - Ответ на сообщения
  - Повторные нарушения
  - Обновления политики

## Стилизация и дизайн

### Цветовая схема
- **Основной фон**: `#0f0f23` (темно-синий)
- **Акцентный цвет**: `#42b883` (зеленый)
- **Вторичный акцент**: `#646cff` (синий)
- **Текст**: `#ffffff` (белый)
- **Вторичный текст**: `#9ca3af` (серый)

### Адаптивность
- **Desktop**: > 1024px
- **Tablet**: 768px - 1024px
- **Mobile**: 480px - 768px
- **Small Mobile**: 360px - 480px
- **Very Small**: < 360px

### Компоненты дизайна
- **Стеклянный эффект**: `backdrop-filter: blur(10px)`
- **Градиенты**: для кнопок и акцентов
- **Скругленные углы**: `border-radius: 8px-16px`
- **Тени**: для карточек и модальных окон
- **Анимации**: плавные переходы и hover-эффекты
